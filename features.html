Pseudo Code
  • We need a funtion that will take in two numbers as parameters (we will call these two numbers [paramOne] and [paramTwo])
  • We are going to need a range of numbers from 1 to 100 and we can use a for loop to get that range
  • We will need to figure out if that number in the index (we will call this [i]) in the for loop is divisible by the [paramOne], [paramTwo], or both
  • with the remainder being zero, check if [i] is divisible by both parameters push the word "FizzBuzz" to an array, if [i] is divisible by only [paramOne] push the word "Fizz" to the array, if [i] is divisible only by [paramTwo] push the word "Buzz" to the array, and if neither of the three criteria checks out just push the number to the array
  • We are going to need an array to contain these values

    For Loop

    Everything is a for loop. we will loop through numbers 1 to 100 and check to see if each of those numbers meets the criteria

  • Modulus

    I think is it time to look at an important tool that will help us in our check to see if our number is divisible by three and five.

    The modulus operator (%) returns the division remainder. (W3 Schools)
    var x = 5;
    var y = 2;
    var z = x % y;
    z == 1
    However we want the remainder to equal zero in order to find out if [i] is divisible by three or five. So our if statement will look a little bit like this:
    if (i % 3 == 0) { return ("Fizz") }
